# GameIQ Changelog

## v2.0.0 - Launch Update (February 2026)

### New Games
- **Trend** - See half a graph (unlabeled), predict direction (up/down/flat), reveal real data. 5 rounds per day.
- **Rank** - 5 items shown, drag to reorder by hidden metric (population, GDP, etc.). 1 puzzle per day.
- **Crossfire** - One word satisfies two clues from different domains. Text input, 5 rounds per day.

### Removed
- **Odd Angle** - Removed from the game roster

### PWA Setup
- Installed `vite-plugin-pwa` for service worker generation
- Generated PNG icons from SVG:
  - `public/icons/icon-192.png` (192x192)
  - `public/icons/icon-512.png` (512x512)
  - `public/icons/apple-touch-icon.png` (180x180)
- PWA manifest auto-generated by Vite plugin
- Workbox service worker for offline caching
